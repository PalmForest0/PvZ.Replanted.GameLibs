# Game Assemblies Library for PvZ: Replanted
Reference this package to get access to the unhollowed assemblies of PvZ: Replanted.

## Installation
You can easily add the Game Libraries to any [BepInEx](https://builds.bepinex.dev/projects/bepinex_be) or [MelonLoader](https://melonwiki.xyz/#/) plugin by simply running:

```bash
dotnet add package PvZ.Replanted.GameLibs
```

This package is included in my BepInEx plugin template for Plants vs Zombies: Replanted, which you can get [here](https://github.com/PalmForest0/Replanted.PluginTemplate).
